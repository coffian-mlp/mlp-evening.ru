# 📜 Лог чейнджлингов (Changelog)

Все важные изменения MLP-Evening заносятся в этот свиток верной ученицей Твайлайт Спаркл.

Формат вдохновлён [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/), версии следуют
принципу: **`X.0.0` / `X.Y.0`** — тематическая веха с именем, **`X.Y.Z`** — хотфикс или патч.
Каждой записи соответствует git-тег на релизном коммите.

## v4.6.0 - Идеальный порядок (16.07.2026)
Наводим идеальный порядок в замке: надёжность, хвост обороны и гигиена свитков и ассетов.

*   **🛡️ Хвост обороны:** Панель управления базой теперь пускает только к настоящим таблицам и колонкам (сверка со схемой — никаких хитрых имён). Пароли — не короче 8 и не длиннее разумного. Токены живого чата (Centrifugo) получили срок годности. Изменять и удалять события можно только с «Печатью Гармонии» (CSRF).
*   **🔒 Вебрут под замком:** Служебные свитки (миграции, тесты, исходники, дампы) больше не подглядеть снаружи — закрыты и на уровне Apache, и на уровне nginx.
*   **🗂️ Журнал заклинаний:** Появился учёт применённых миграций (`schema_migrations` + `migrate.php`) — теперь замок помнит, какие свитки уже накатаны. И чат стал шустрее на выборках по пользователю (новый индекс).
*   **⚡ Кеш снова дышит:** Стили и скрипты кешируются по-умному (версия по времени файла, а не «каждый раз заново»), а Service Worker обновляет свой кеш при каждом релизе.
*   **🤖 Порядок у Лиры:** В режиме демона Лира теперь сразу видит правки настроек из Дашборда. Команды бота перестали пересоздаваться при каждом обращении — у них появился свой хранитель (`BotCommandManager`).
*   **🧩 api.php худеет (начало):** События переехали из гигантского свитка в тонкий роутер с отдельным хранителем расписания (`EventManager`). Первый шаг большого наведения порядка — дальше по чуть-чуть.

## v4.5.0 - Долгая память (13.07.2026)
Эпоха надёжности: помним пони дольше, чиним чат и ускоряемся.

*   **💜 «Запомни меня»:** Галочка при входе — и Эквестрия не выкинет тебя из чата, пока сама не захочешь уйти. Сессия живёт долго и продлевается при каждом возвращении, а вход через соцсети запоминается сам собой. (Токены хранятся безопасно и отзываются при выходе.)
*   **📡 Починка «коробочного» чата (SSE):** Восстановлена живая доставка сообщений в SSE-режиме — он снова работает без Centrifugo (который по-прежнему рекомендуется для больших залов).
*   **⚡ Быстрее настройки:** Глобальные настройки читаются одним запросом вместо десятков — админка и страницы отзывчивее.
*   **🧹 Чище код обороны:** Проверка прав администратора вынесена в единую точку (меньше шансов где-то забыть замок).

## v4.4.1 - Щит Гармонии: завершение (13.07.2026)
Финальный штрих к обороне замка.

*   **🔒 Крепче сессии:** Куки сессии получили защитные флаги, а при входе выдаётся новый идентификатор — чужую сессию больше не «подсадить».
*   **🌐 Загрузка по ссылке — только наружу:** Загрузка картинок по URL теперь ходит лишь на внешние публичные адреса (внутренняя сеть под запретом).
*   **🤫 Меньше болтовни об ошибках:** Технические детали ошибок больше не показываются гостям — уходят в лог.
*   **🔑 Точнее проверка входа через Telegram:** Сверка подписи стала строже и безопаснее.

## v4.4.0 - Щит Гармонии (13.07.2026)
Обновление безопасности. Наводим порядок в защите замка, пока злодеи не подобрались.

*   **🛡️ Укрепление панели настроек:** Изменять глобальные настройки сайта теперь может только Принцесса (администратор) — доступ строго проверяется на стороне сервера.
*   **🗂️ Защита управления плейлистом и историей:** Операции над плейлистом, голосами и журналом просмотров доступны только администратору (серверная проверка прав).
*   **🧹 Чистка вебрута:** Из публичного доступа убраны служебные и отладочные скрипты; фоновые процессы бота (воркер) теперь запускаются только из командной строки, но не через браузер.
*   **🔐 Секреты — под замок:** Служебные бинарники и конфиги с секретами убраны из репозитория (и из всей его истории); заклинания хранятся отдельно от кода.

## v4.3.0 - Пробуждение Лиры (13.07.2026)
Большое обновление AI-магии и полировка чата.

*   **🤖 Очередь Лиры (BOT-QUEUE):** Ответы бота теперь готовятся в фоновой очереди (`llm_jobs`) единым воркером. Чат больше не ждет, пока Лира подберет слова — даже если её зовут сразу несколько пони! Режимы воркера: daemon / cron / inline с умной деградацией.
*   **👁️ Зрение:** Лира научилась видеть картинки в чате (vision-модели, аккуратные base64-миниатюры вместо ссылок, и только из свежих сообщений — экономим магию).
*   **🌐 Новые источники магии:** Провайдер RouterAI (routerai.ru) и VLESS-прокси (Xray) для доступа к заморским моделям.
*   **🔒 Защита заклинаний:** Системные инструкции бота больше не просачиваются в чат; Лира отвечает только когда к ней действительно обращаются.
*   **🎉 Больше эмоций:** Новые реакции — Сердечко, Огонь, Вау, Задумчивость, Вечеринка, Крутость (всего 11!). И да, Лира тоже ставит реакции!
*   **📱 Мобильный чат:** Поле ввода переработано под стать десктопу (постоянная нижняя панель), пикер реакций больше не убегает за экран, профиль открывается из попап-чата.

## v4.1.0 - Магия Эмоций и Улучшений (20.02.2026)
Большое обновление, направленное на социальное взаимодействие и удобство чата.

*   **🧩 Модульная Архитектура:** Полный рефакторинг ядра на компонентную систему (**Bitrix-like**). Все блоки (чат, плеер, админка) теперь живут в изолированных компонентах.
*   **🎉 Реакции на сообщения:**
    *   Возможность ставить эмодзи-реакции (Лайк, Дизлайк, Смех и др.).
    *   Smart Tooltip: При наведении видно, кто поставил реакцию (с аватарками!).
    *   Инлайновое отображение для экономии места.
*   **🔍 Поиск и Навигация:**
    *   Поиск по истории сообщений (кнопка Лупы).
    *   "Прыжок во времени": Загрузка контекста старого сообщения при клике на цитату или результат поиска.
*   **💬 Умное Цитирование:**
    *   Выдели текст -> Нажми "Цитировать" -> Получи красивую цитату выделенного фрагмента.
    *   Подсветка сообщений, являющихся ответами вам.
*   **🖼️ Медиа (Lightbox):**
    *   Полноценный просмотр картинок на весь экран с зумом (`Ctrl+Wheel`) и панорамированием.
*   **🛠️ Администрирование:**
    *   **Debug Mode:** Переключатель отображения ошибок PHP в дашборде.
    *   Улучшенная защита от дублей сообщений и "распухания" поля ввода.
*   **📱 UX/UI:**
    *   Ghost Button: Компактная кнопка отправки внутри поля ввода.
    *   Мягкая авторизация (без перезагрузки страницы).
    *   Случайные цвета для новых пользователей.
