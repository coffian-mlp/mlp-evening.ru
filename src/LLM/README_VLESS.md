# 🦄 Настройка VLESS для Живой Пони

Если ты хочешь использовать мощные зарубежные модели (OpenAI, Groq, OpenRouter), но твой сервер находится в зоне блокировок, и у тебя есть ссылка вида `vless://...`, наша Живая Пони может сама пробить этот туннель!

Чтобы эта магия сработала "из коробки" (через кнопку в админке), тебе нужно просто положить "лопату" для этого туннеля рядом с кодом.

## Инструкция:
1. Скачай официальный бинарник **Xray-core** под систему своего сервера (например, `Xray-linux-64.zip`) с [официального GitHub-репозитория](https://github.com/XTLS/Xray-core/releases/latest).
2. Распакуй архив.
3. Положи файл `xray` (или `xray.exe`, если у тебя Windows) в эту папку: `src/LLM/bin/xray`.
   *Убедись, что у файла есть права на исполнение (`chmod +x src/LLM/bin/xray`).*
4. Зайди в Админку своего сайта -> Настройки ИИ Бота.
5. Вставь свою `vless://...` ссылку прямо в поле "Прокси".
6. Сохрани настройки.

Всё! При первом же вызове нейросети PHP сам запустит этот Xray в фоне, сгенерирует для него правильный конфиг и пустит весь трафик через него. Если что-то пойдет не так (например, прокси умрет), Пони не зависнет, а просто попросит ответа у Яндекса или Сбера.
